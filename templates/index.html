About Us

The automatic generation of presentation slides from technical articles is one of the most desired but under-researched area in the field of computing.Presentation slides have been widely used over a very long time for the abstract conveyance of ideas. It helps users to transfer their ideas and theories effectively.We provide you the User interface to Select the particular files (ie.whole files or part of the files(ie.paragraph). User can have their only summerzation of the input or Presentation, decision will be taken based on the user Reqirements.

Services

  • Step 1

    User Input

    Reading the input from the user by providing the web Interface

  • Step 2

    Text Conversion

    Once the file read from the user, it is converted into a text file, so that file can be used to the summarization.!

  • Step 3

    Text Summarization

    Converted text file is used to generate the summerizatio using NLP Techniques..(In this Phase user can directly use summerization as their output, if user requires the presentation,please Follow the step 4!

  • Step 4

    Presentation Building

    This is the final step, Summerized text is used to build the presentation Automatically.
